# Responsive-JavaScript-Based-Quiz-Application
Built an interactive web-based quiz app using HTML, CSS, and JavaScript. Implemented features like dynamic question loading, score  calculation, real-time feedback, and a countdown timer. Designed a responsive user interface optimized for mobile and desktop screens.

## Features
- Responsive UI
- Timer-based questions
- Instant answer feedback
- Score tracking


## How to Run
Open `index.html` in a browser or deploy using GitHub Pages.
